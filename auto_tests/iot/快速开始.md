# ⚡ 快速开始

## 第一步：确认环境

确保已安装 Python 3.8+ 和 Google Chrome 浏览器。

打开终端，执行：
```bash
python --version
pip install selenium webdriver-manager openpyxl
```

---

## 第二步：运行测试

**Windows 用户（推荐）**：双击 `run_test.bat`

**命令行用户**：
```bash
cd auto_tests/iot
python device_stats_ui_automation.py
```

---

## 第三步：输入验证码

脚本启动后会自动：
1. 打开 Chrome 浏览器
2. 导航到 https://iot.csmart-test.com/#/login
3. 填写账号（raolekang）和密码

🔴 **此时需要你手动**：在浏览器的验证码框中输入图形验证码

> 脚本最多等待 **15 秒**，检测到输入后自动继续，无需按任何键。

---

---

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| 登录失败 | 检查 login_failed_*.html，确认验证码是否正确输入 |
| 查询按钮未找到 | 确认已正确切换至市场化盘，检查 before_filter_*.html |
| 4G手表点击失败 | 图表由 ECharts 渲染，脚本会自动尝试 JS 触发，可查看日志 |
| ChromeDriver 报错 | 执行 `pip install --upgrade webdriver-manager` |
