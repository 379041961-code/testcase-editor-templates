# 快速开始

## 第一步：确认环境

- 已安装 Python 3.8+。
- 已安装 Google Chrome 浏览器。
- 依赖包可通过以下命令安装：

```bash
pip install selenium webdriver-manager openpyxl
```

## 第二步：运行脚本

- 推荐方式：双击 run_test.bat。
- 命令行方式：

```bash
cd auto_tests/readme/device_stats_ui
python device_stats_ui_automation.py
```

## 第三步：输入验证码

- 脚本会自动打开登录页面并填写账号密码。
- 请提前设置环境变量：

```bat
set IOT_TEST_USERNAME=你的账号
set IOT_TEST_PASSWORD=你的密码
```

- 验证码需要手动输入，脚本自动轮询检测，最多等待 15 秒，无需按键确认。

## 常见问题排查

| 问题 | 解决方法 |
|------|----------|
| 提示缺少依赖 | 执行 pip install selenium webdriver-manager openpyxl |
| 登录失败 | 检查验证码是否在 15 秒内输入，确认账号密码环境变量已设置 |
| 页面元素定位失败 | 查看 html 目录快照，确认测试环境页面结构是否变化 |
| ChromeDriver 报错 | 执行 pip install --upgrade webdriver-manager |
